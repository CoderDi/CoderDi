Услуги

Выполняю следующие работы
Вёрстка
Верстка HTML-сайта (лендинги, многостраничные сайты интернет-магазины) из макета Photoshop, Figma, Adobe XD.
Валидно, кроссбраузерно, адаптивно.
Полная работоспособность в HTML Вашего макета.
Дизайн
Разработка дизайна по Вашему прототипу и ТЗ либо “сделай как у конкурентов, только лучше”. Конечным продуктом, являются макеты Figma, готовые для верстки шаблона.
Сайт на WordPress
Создание сайта на Wordpress как по вашему дизайну и верстке, так и с нуля под ключ.
Наполнение сайта конентом - текстовой, графической информацией.
Скрипты и доработка
1. Установка, доработка дополнительного функционала: калькулятор, фотогалерея, слайдер, карта сайта, формы для связи и т.д.
2. Оптимизация сайта: скорость загрузки, оптимизация сайта для лучшей индексации поисковыми системами: HTML и CSS валидация, корректировка кода и пр.
3. Наполнение сайта Вашим контентом: товарами интернет-магазина, текстовой информацией, графической информации.
4. Регистрация домена, хостинга, установка Вашего сайта на хостинг.
5. Администрирование и сопровождение Вашего сайта.
